Mockapitron

Tool for generating random api response based on

and

What do you need to get started ?

Send POST request to http://api-mockapitron.rhcloud.com/ with schema and size parameters

Default values for parameters { schema:{}, size:0 }

Usage example:

fetch('http://api-mockapitron.rhcloud.com/', {
  method: 'post',
  headers: {
    'Accept': 'application/json',
    'Content-Type': 'application/json'
  },
  mode: "cors", 
  body: JSON.stringify({
    schema: {
      name: "name",
      address: "address"
    },
    size: 3
  })
})
.then(function(res){ return res.json() })
.then(function(data){ console.log(data) })

Standard schema property types:

  • String
  • Number
  • Date
  • Boolean
  • Array

Few custom schema property types:

  • Name
  • Age
  • Address
  • Gender
  • (RGB) color
